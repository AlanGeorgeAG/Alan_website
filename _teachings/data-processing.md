---
layout: course
title: Data Processing
description: A practice-oriented course on transforming raw data into reliable analysis-ready datasets for scientific and machine learning workflows.
audience: Bachelor's and master's students
course_type: Teaching
teaching_format: Concept-driven instruction with applied workflow examples
order: 1
topics:
  - Data cleaning
  - Preprocessing
  - Transformation
  - Pipelines
  - Feature engineering
  - Validation
modules:
  - title: Foundations of clean data
    description: Students learn how to inspect data quality, identify structural issues, and prepare raw inputs for downstream analysis.
    topics:
      - Missing-value handling and consistency checks
      - Outlier inspection and error detection
      - Data typing, normalization, and standardization
  - title: Transformation and workflow design
    description: The course emphasizes reproducible processing chains that support later statistical or machine learning analysis.
    topics:
      - Tabular restructuring and feature formatting
      - Transformation pipelines and staged preprocessing
      - Building robust inputs for model training
  - title: Feature preparation and interpretation
    description: Students connect preprocessing choices to model behavior and scientific interpretation.
    topics:
      - Feature engineering strategies
      - Leakage-aware preparation of training data
      - Interpreting how preprocessing changes downstream results
---

This course is designed to make students comfortable moving from messy raw data to structured, defensible datasets. It focuses on the practical decisions that influence reproducibility, model quality, and scientific interpretation.
