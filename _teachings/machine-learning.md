---
layout: course
title: Machine Learning
description: An application-focused introduction to machine learning methods, model development, and critical evaluation for scientific datasets.
audience: Bachelor's and master's students
course_type: Teaching
teaching_format: Conceptual lectures paired with practical modeling workflows
order: 2
topics:
  - Supervised learning
  - Unsupervised learning
  - Regression
  - Classification
  - Clustering
  - Model evaluation
modules:
  - title: Supervised learning workflows
    description: Students build intuition for predictive modeling tasks and the assumptions behind common algorithms.
    topics:
      - Regression for continuous targets
      - Classification for categorical outcomes
      - Training, validation, and generalization
  - title: Unsupervised learning and structure discovery
    description: The course covers exploratory learning methods that reveal patterns without labeled targets.
    topics:
      - Clustering and grouping structure
      - Distance metrics and representation choices
      - Interpreting latent patterns in data
  - title: Evaluation and interpretation
    description: Strong emphasis is placed on understanding why a model performs the way it does and how to communicate results responsibly.
    topics:
      - Validation strategies and performance metrics
      - Error analysis and model comparison
      - Practical interpretation of machine learning outputs
---

The course balances methodological foundations with practical judgement. Students learn not only how to train models, but also how to decide whether a model is trustworthy, appropriate for the data, and useful for real analytical questions.
